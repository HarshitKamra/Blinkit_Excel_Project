# Blinkit Sales Analysis Dashboard | Microsoft Excel

An interactive sales dashboard built in Microsoft Excel using a publicly available Blinkit sales dataset. The project focuses on transforming raw retail data into meaningful business insights through data cleaning, KPI reporting, and interactive visualization.

---

## Project Overview

Retail businesses generate large volumes of sales data every day. The objective of this project was to analyze a Blinkit sales dataset containing **8,523 retail records** and build an interactive dashboard that enables stakeholders to monitor performance and make data-driven business decisions.

The dashboard provides insights into sales performance across outlet locations, outlet sizes, product categories, and establishment years while allowing dynamic filtering using slicers.

---

## Business Objectives

This dashboard was built to answer questions such as:

- Which outlet locations generate the highest revenue?
- Which product categories contribute the most to total sales?
- How does outlet size impact sales performance?
- Which outlet types perform the best?
- What insights can support inventory and expansion decisions?

## Key Insights

- Tier 3 outlets generated the highest overall sales.
- Fruits & Vegetables emerged as the highest-performing product category.
- Regular-fat products contributed a larger share of revenue than low-fat products.
- Medium and High-sized outlets accounted for the majority of total sales.

---

## Business Recommendations

- Increase inventory allocation for high-performing Tier 3 outlets.
- Prioritize inventory for top-selling product categories.
- Strengthen promotional efforts for lower-performing categories.
- Use outlet-level sales insights to support inventory planning and future expansion.
